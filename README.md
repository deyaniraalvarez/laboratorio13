# Laboratorio 13 - Calculadora básica con funciones en JavaScript

## Objetivo
Implementar una calculadora básica en JavaScript que utilice funciones para operaciones matemáticas y una función principal que reciba parámetros.

## Archivos
- index.html - Interfaz simple para ejecutar la calculadora en el navegador.
- calculator.js - Funciones matemáticas y la función calculadora.
- node_calculator.js - Versión para ejecutar desde Node.js (terminal).
- .gitignore - Entradas a ignorar.

## Cómo probar
### En el navegador:
1. Abrir index.html en un navegador.
2. Clic en "Ejecutar calculadora (prompt)" y seguir los prompts.
3. Ver resultado en alert y en la consola (DevTools).

### En Node.js:
1. Tener Node instalado.
2. Ejecutar: node node_calculator.js

## Notas
- La función dividir lanza un error si se intenta dividir por cero.
- La función calculadora acepta operaciones como suma, +, resta, -, multiplicar, *, division, /.
